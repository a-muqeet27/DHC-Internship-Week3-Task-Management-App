# 📝 TaskFlow - Flutter Task Management App

## 📌 Overview

TaskFlow is a modern and elegant **Task Management App** built using **Flutter** and **Dart**. It demonstrates Flutter fundamentals such as UI design, animations, state management, and local data persistence using `SharedPreferences`.

---

## 📱 Features

- ✅ Add new tasks
- ✅ Delete tasks
- ✅ Mark tasks as completed
- ✅ Save tasks locally using `SharedPreferences`
- ✅ Progress card showing completed tasks
- ✅ Beautiful gradient UI
- ✅ Smooth fade animation
- ✅ Material 3 design
- ✅ Responsive mobile layout

---

## 🧰 Technologies Used

- Flutter
- Dart
- SharedPreferences
- Material Design 3
- JSON Encode / Decode
- AnimationController
